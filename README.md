# Azkarify Husn CDN

Static JSON payloads for Azkarify, intended to be served over jsDelivr from a public GitHub repository.

## Structure

- `v1/ar/husn_ar.json`
- `v1/en/husn_en.json`
- `v1/ar/<id>.json`
- `v1/audio/manifest.json`
- `v1/manifest.json`

## jsDelivr

After publishing this repository publicly, files can be accessed with:

`https://cdn.jsdelivr.net/gh/<owner>/<repo>@<ref>/v1/manifest.json`
